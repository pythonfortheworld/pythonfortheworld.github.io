Working with video in HTML5.


Step 1:  Create a new folder on your Desktop and name it Videos.

Step 2:   Download the youtube-dl.exe application from here and copy it from your Downloads folder to your Videos folder.

Step 3:  Go to Youtube and find any CSS for beginners video and copy the URL.

Step 4: Now go to the command prompt (Start button --> cmd) and type this code:

cd Desktop

cd Videos

youtube-dl "https://www.youtube.com/watch?v=qKoajPPWpmo" ----Note: Use your URL instead of the one you see here! You just need to right-click and paste the URL you copied on the previous step. Do not forget to type the quotes before and after the URL.

Step 5:   Rename the downloaded video as css.mp4.

Step 6:   Create a new file named Wednesday.html in the Videos folder. Add some boiler plate code like the one shown right here .

Step 7:   Now write code within the body to show the video you downloaded!
